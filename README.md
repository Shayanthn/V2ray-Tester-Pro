## ğŸš€ Getting Started

### Prerequisites

- Python **3.8+**
- `pip` installed

> **Note:** `xray` binaries are **already included** in this repo.

### Installation

```sh
git clone https://github.com/shaynthn/v2raytesterpro.git
cd v2raytesterpro
pip install -r requirements.txt
````

**requirements.txt** should contain:

```
aiohttp
psutil
requests
python-dotenv
PyQt6
rich
python-telegram-bot
```
